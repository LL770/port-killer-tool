# 端口进程管理工具

一个简单易用的 Windows 批处理脚本，可以快速关闭占用指定端口的进程。

## ✨ 功能特点
- 支持自定义端口号
- 无需安装，下载即用
- 交互式界面，使用简单

## 🚀 使用方法

### 下载
点击右上角绿色的 "Code" 按钮，选择 "Download ZIP"，解压即可。

### 运行
1. 右键点击 `killport.bat`
2. 输入要关闭的端口号（如 8080），直接回车则默认关闭 8080 端口
3. 按提示完成操作
